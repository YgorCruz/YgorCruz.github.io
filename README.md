# Portfólio - Ygor da Cruz Boueri de Souza

Portfólio pessoal desenvolvido para apresentar meu perfil profissional, minha trajetória, conhecimentos técnicos e uma página dedicada aos projetos publicados no GitHub.

O site foi construído com foco em clareza, visual clean e navegação simples, sem uso de frameworks.

## Páginas

- `index.html`: página principal com apresentação, sobre, conhecimentos técnicos, experiência, formação e contato.
- `projetos.html`: página dedicada aos projetos públicos do GitHub, com resumo de cada repositório.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Estrutura do projeto

```text
portifolio/
├── Css/
│   └── Style.css
├── Img/
├── Js/
│   └── main.js
├── index.html
├── projetos.html
└── curriculo.pdf
```

## Como visualizar localmente

Você pode abrir o arquivo `index.html` diretamente no navegador ou usar uma extensão como Live Server no VS Code para navegar entre as páginas.

## Objetivo

Este projeto foi criado para consolidar minha presença profissional online e reunir, em um só lugar:

- apresentação pessoal
- experiência profissional
- formação acadêmica
- conhecimentos técnicos
- projetos públicos no GitHub
